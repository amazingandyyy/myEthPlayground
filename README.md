# myEthPlayground

feel free to hangaround and check any contracts and test file.

## clone down and prepare
```
$ npm install -g ethereumjs-testrpc truffle
```


## Start to use it

one tab in terminal
```
$ testrpc
```

new tab in terminal, in the folder->
```
$ truffle deploy
$ truffle test
```
